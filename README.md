# Movie Browser

## Requirements
 1. Have NodeJS and Git installed.
 2. Install `gulp` and `jest` globally:
 ```
    npm install -g gulp jest
 ```
 3. Clone the package and install the local dependencies:
 ```
    git clone TBD
    npm install
 ```

## Commands
 * Build sources: `gulp`
 * Clean build files: `gulp clean`
 * Run tests: `jest`
